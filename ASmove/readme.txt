Перенос проекта AS
1. Системы контроля версий (Git/GitHub) — Рекомендуемый
Это самый надежный способ, позволяющий синхронизировать код и работать над ним с разных устройств. 

    На первом ПК: В меню Android Studio выберите VCS -> Share Project on GitHub (или создайте репозиторий локально и сделайте push на Bitbucket/GitLab).
    На втором ПК: Выберите File -> New -> Project from Version Control и укажите ссылку на ваш репозиторий.
