
1)Из cmd admin установить глобально expo-cli:
npm install -g expo-cli
2) Создать проект
   Перейти в папку для проектов:
     a. Cd ИМЯ_ПАПКИ_ВСЕХ_ПРОЕКТОВ
   Создать новый проект. Папка для проекта будет создана:
     b. npx create-expo-app ИМЯ_ПРОЕКТА_И ПАПКИ
3) Войти 
	a. в Visual Studio 
	b. открыть проект ИМЯ_ПРОЕКТА_И ПАПКИ
4) Запустить на выполнение:
   a. Открыть терминал и
   b. npx expo start
	Возможно понадобится: 
	c. expo install react-native-web react-dom @expo/metro-runtime
   d. IP под QR кодом указать в браузере
5) Скопировать содержимое GitHubLab в свой проект и обновить браузер
6) в дальнейшем изменения будут в папке lab.
